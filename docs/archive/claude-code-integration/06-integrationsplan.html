③Voice-Dispatch — der WhisperM8-Move

Drückst du den globalen Hotkey, wenn kein Tab den Fokus hat, kannst du einen neuen Hintergrund-Agent direkt per Stimme spawnen. Der existierende Recording-Overlay bekommt einen zusätzlichen Modus → Background-Agent; der Transkript-Text läuft in claude --bg "<text>" + optional --agent <name>. Toast bestätigt den Spawn, Sidebar zeigt die neue Row.

④Inline-Approvals — Hooks statt PTY

Phase 3 · Lifecycle

Subprocess-Wrapper

  • Detach / Re-Attach / Logs / Stop / Respawn / Remove

Phase 4 · Hook-Bridge expand

⑧Risiken & Mitigations

RisikoMitigation
Stdout-Format von claude --bg ändert sich Defensiver Regex, Fallback: roster.json vor/nach Spawn diffen — neue Short-ID = die mit höchster created_at
Organisations-Policy disableAgentView: true Beim Onboarding claude --bg --help probe-checken; Feature-Flag, BG-Funktionen grayed out anzeigen mit Erklär-Tooltip
--permission-mode bypassPermissions ohne Konsens blockiert Onboarding-Step: einmal interaktiv durchführen lassen; danach in Settings persistent als "akzeptiert" markieren
User hat CLAUDE_CONFIG_DIR gesetzt Bei jedem Indexer-Run die Env-Var lesen + Pfade dynamisch resolven
Permission-Approval via Sidebar reicht nicht in PTY-State zurück Phase 4.1: Eigenes permission-prompt-tool MCP-Server bauen; oder per claude attach stdin-Bridge antworten
Auto-Update von Claude Code bricht eine Version claude --version bei Start parsen, Feature-Gate auf min. 2.1.139; Warning-Banner wenn unter Min-Version

⑨Was am Ende dasteht

🎙 Voice-Driven Agent Orchestrator
"Sprich Tasks an, WhisperM8 dispatched sie als Background-Agents, kriegt Push-Toasts wenn einer Input braucht, mergst PRs mit Klick." Eine UX, die sonst niemand anbietet.
🖥 Native macOS-Alternative zur Anthropic-TUI
Vollständig in Swift gerenderte Multi-Session-Übersicht, mit Maus, Drag, Cmd+Klick, Right-Click, Multi-Select — alles, was die TUI nicht kann.
🪝 Hook-driven Live-Status ohne API
Anthropic gibt kein offizielles Read-API für Agent-View-State. Wir nutzen Hooks (offiziell, stabil, file-watch-basiert mit 0 % Idle-CPU) und sind unabhängig von Implementation-Details.
🔁 Sub-Agent-Library als visuelle Erfahrung
Sub-Agents werden zu first-class Citizens: erstellen, editieren, in Dispatch-Modal auswählen — der Sub-Agent-Layer von Claude Code wird in macOS sichtbar gemacht.
📦 Worktree-, Branch- & PR-Awareness
Background-Agents arbeiten in .claude/worktrees/. WhisperM8 zeigt den Worktree, integriert "öffnen in IDE", PR-Status, "Mergen wenn grün". Das macht aus jedem Agent ein verschiffbares Ergebnis.
